# AGENTS.md — Контекст для AI-агентов

## Что это
Веб-карта скалодромов Буэнос-Айреса. Одностраничное приложение (SPA) без бэкенда.

## Решения
- **2026-02-21**: Создан проект по запросу Сереги. Выбран Leaflet + OSM (бесплатно, без API ключей). Один HTML файл для простоты.
- Координаты скалодромов взяты из Google Maps / индексов climbing-gyms.com и indoorclimbing.com.
- Палестра Nacional (CeNARD, Núñez) НЕ включена — снесена в дек 2025, новая строится в Parque Olímpico.

## Как добавить скалодром
В массив `gyms` в index.html добавить объект:
```js
{
  name: "Название",
  address: "Адрес",
  lat: -34.XXXX, lng: -58.XXXX,
  schedule: "Расписание",
  ig: "@instagram",
  features: "Описание"
}
```
